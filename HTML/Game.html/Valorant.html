FAVOURITE GAME : VALORANT.

Brimstone:-

  1. Smoke: You can select one or multiple points on a map that drops a smoke grenade that obscures vision.

  2. Orbital Strike (ultimate): Select an area on the map and a laser damages all inside the blast radius.

  3. Stim Beacon (optional): An AOE combat stim which increases fire rate. Both teammates and enemies alike can use this.

  4. Incendiary (optional): A grenade launcher that fires off an incendiary round.

Raze:-

  1. Paint Shells: Throws a cluster grenade. Explodes and deals damage.

  2. Showstopper (ultimate): Uses a rocket launcher that deals damage over a large area.

  3. Boom Bot (optional): Walks across the floor and later explodes, dealing damage to enemies.

  4. Blast Pack (optional): Throws like Semtex grenades or C4. Can be triggered.

Cypher:-

  1. Spycam: Throws a camera down that can signal where opponents are.

  2. Neural Theft (ultimate): Throws down a device on dead opponents that can show where the enemy team is.

  3. Trapwire (optional): A tripwire that slows enemies and reveals their location — invisible to enemy eyes.

  4. Cyber Cage (optional): Drops an AOE screen that slows opponents, obscures vision and is signaled by an audio cue.
​

Jett:-

  1. Tailwind: Dash in the direction you're moving.

  2. Bladestorm (ultimate): Get several knives that you can throw at opponents — recharges as you get kills.

  3. Cloudburst (optional): Throw a projectile that will throw a vision-blocking cloud.

  4. Updraft (optional): You can float up to otherwise-unreachable heights and float around.
